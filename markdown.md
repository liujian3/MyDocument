# markdown基本语法

## 目录  

[一、标题](#一标题 "一、标题")  
[二、字体](#二字体 "二、字体")



## 一、标题

```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

效果如下:  

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

## 二、字体  

```
*这是倾斜的文字*`
**这是加粗的文字**
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```

效果如下:  

*这是倾斜的文字*`  
**这是加粗的文字**  
***这是斜体加粗的文字***  
~~这是加删除线的文字~~
